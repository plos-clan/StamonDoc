# 字节码

该文档详细讲述了Stamon字节码文件的组成, 以及各指令的功能

如果有能力, 您也可以依据此文档自行实现一个可以编译Stamon标准语法的编译器

# 结构

一个标准的Stamon字节码文件应当符合以下结构

Index(字节) ByteCodeFile
1 - 2 0xADBD 文件魔数
3 - 4 Stamon字节码版本号
5 - 6 常量表总项数
(7+n) 常量表 - 详见常量表
(7+n) + a 指令区

# 常量表

用于存储脚本运行过程中所需的所有值,标识符等. 作为部分字节码指令的操作数

为了增加VM索引速度, 规定表项第一位是存储了标识符类型的__init__值

以下是一个常量表各种类型项的组成

# 标准整形

Index(字节) BinaryData
n 0x01
n+2 data

# 双精浮点

Index(字节) BinaryData
n 0x02
n+8 data

# 字符串值

Index(字节) BinaryData
n 0x03
n+2 Length
n + Length data

# 标识符

Index(字节) BinaryData
n 0x04
n+2 Length
n + Length data

# 无参指令

单字节指令,不包含任何操作数

助记符 指令 描述
ADD 0x01 将操作栈栈顶两元素相加
SUB 0x02 将操作栈栈顶两元素相减
CPL 0x03 将栈顶元素按位取反
NOT 0x04 将栈顶元素进行逻辑非运算
NEW 0x05 根据栈顶元素创建实例对象
MUL 0x06 将栈顶两元素相乘
DIV 0x07 将栈顶两元素相除
MOD 0x08 栈顶两元素取余
LSH 0x09 栈顶元素位左移运算
RSH 0x0A 栈顶元素位右移运算
LESS 0x0B 栈顶两元素小于关系比较
LEQ 0x0C 栈顶两元素小于等于比较
BIG 0x0D 栈顶两元素大于关系比较
BEQ 0x0E 栈顶两元素大于等于比较
EQU 0x0F 栈顶两元素等于关系比较
IEQ 0x10 栈顶两元素不等于比较
BAND 0x11 栈顶两元素位与运算
BXOR 0x12 栈顶两元素位异或运算
BOR 0x13 栈顶两元素位或运算
LAND 0x14 栈顶两元素逻辑与运算
LOR 0x15 栈顶两元素逻辑或运算
ASSIGN 0x16 将栈顶元素赋值到栈下层元素
MEMBER 0x17 从栈顶元素提取成员
CALL 0x1E 执行方法
SFN 0x20 SFN本地接口指令
POS 0x21 栈顶元素取正
NEG 0x22 栈顶元素取负
RET 0x23 返回栈顶元素
ARRAY 0x24 加载数组

保留指令
BRK 0x25 终止循环
CTN 0x26 退出此次循环

# 有参指令

长度一般为2字节的指令,通常包含一个指令和操作数

index 常量表索引
pc 指令索引
label 标记点

助记符 用法 指令 描述
DEF def index 0x18 加载变量到变量表
GOTO goto pc/label 0x1c 无条件跳转到指定位置
PUSH push index 0x1d 将指定元素压入操作栈
IF if pc/label 0x1b 如果栈顶元素不是0或null,跳转至指定位置
