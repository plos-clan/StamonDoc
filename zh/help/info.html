# 关键字

关键字与保留字: Stamon中用于定义语句、变量、类等所用的字段, 这些字段不能被用作方法名、类名、变量名等

字段 说明
import 用于导入内置库或第三方脚本
func 用于定义方法
class 用于定义类
def 用于定义变量
if If语句定义
else Else语句定义
while 循环语句定义
for 循环语句定义
sfn 调用SFN指令
true 布尔值"真"
false 布尔值"假"
break 循环退出
continue 终止此次循环
null 空值
in 辅助for循环
