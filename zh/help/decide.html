# 判断语句

Stamon中基本的流程控制语句,可通过布尔表达式的值决定是否会指定语句块内的代码.

# If

  • 在Stamon中,使用if关键字定义一个判断语句

括号内的表达式最终结果必须确保为布尔值,
因为Stamon弱类型的特性,CoolPot等一些静态编译器不会做表达式预推断
编译出来的字节码会发生运行时错误

script.st

import std;
if(true){
    println("一个判断语句");
}

# Else

  • 在Stamon中,使用else关键字定义一个否则语句
  • 在else语句前必须有一个if或else if语句的定义

script.st

import std;
if(false){

}else{
    println("否则语句块");
}
