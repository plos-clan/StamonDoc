本教程将讲解如何编写一个简单的Stamon脚本并运行

# 你好！世界！

您需要导入 Stamon 基础库 std 来使用 println 方法

script.st

import std;
println("Hello! World!");

# 注释

在 Stamon 中，支持单行注释//, 也支持多行注释/**/.

script.st

import std; //导入基础库
println("Hello! Stamon!");
/*
println方法定义在stddef.st中
但是std.st已经将所有常用方法都包含进来了
所以非特殊情况您只需要使用import std;来导入即可
*/

# 关键字与保留字

详见 更多信息

注意!

在Stamon中,语句之间必须用;分割,即使该条语句没有下一条语句,您也必须使用;结尾.
对于特殊的func class语法糖, 末尾如果没有表达式参与的话您可以省略;

def语句中所有的词素都算做一整条单个语句,您必须在末尾添加;

def func_name = func(){

}; //必须加上分号

/*
def func_name = func(){

} 不允许这种写法
*/
