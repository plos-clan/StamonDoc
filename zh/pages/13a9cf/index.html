# 基元变量

Stamon中
虽然是隐式类型(这代表你在定义变量时候无需显式指定类型, 变量重赋值也不需要与旧类型统一)
但我们仍然需要注意一些隐藏的类型

script.st

def integer_v = 123;
def boolean_v = true;
def double_v = 3.1415926;
def string_v = "Hello! Stamon!";

# 实例变量

Stamon中
创建一个类的实例对象也是用def关键字

以下是常规的定义方式,与创建实例对象

script.st

def class_name = class {
    def __init__ = func(self){
        // pass
    };
};

def instance = class_name.new();
