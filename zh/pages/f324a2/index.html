# 运行

您可以在任何平台运行StamonVM

# WindowsNT

  1. 从GitHub存储库下载到指定发行版, 请选择*.win32.exe文件至您的系统
    • 前往 GitHub 存储库 (opens new window)
  2. 配置PATH系统环境变量到stamon的根目录中
  3. 启动Windows PowerShell

Windows PowerShell

PC> stamon -h
1
